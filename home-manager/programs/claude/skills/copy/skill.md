Claude Codeへの指示を実行し、その結果をクリップボードにコピーします。

## 使用方法

### 1. 指示を実行してコピー

```
/copy <Claude Codeへの指示>
```

引数で指定された指示を実行し、その結果の出力をクリップボードにコピーします。

### 2. 直前の出力をコピー

```
/copy
```

引数なしで実行すると、直前のツール実行結果やコマンド出力をクリップボードにコピーします。

## 実装手順

1. 引数の有無を確認
2. 引数がある場合:
   - 引数の指示を実行（コマンド実行、ファイル読み取り等）
   - 実行結果の出力を取得
   - 出力を `echo "<出力>" | pbcopy` でクリップボードにコピー
   - コピー完了を報告
3. 引数がない場合:
   - 会話履歴から直前のツール出力を取得
   - その内容を `echo "<出力>" | pbcopy` でクリップボードにコピー
   - コピー完了を報告

## 注意事項

- macOSの `pbcopy` コマンドを使用します
- 長い出力の場合は、そのまま全てコピーされます
- マルチライン出力は改行も含めて保持されます

## 例

### コマンド実行結果をコピー
```
User: /copy git log --oneline -5
→ git log を実行し、その出力をクリップボードにコピー
```

### ファイル内容をコピー
```
User: /copy README.mdの内容
→ README.mdを読み取り、その内容をクリップボードにコピー
```

### コード検索結果をコピー
```
User: /copy src/配下の全てのTypeScriptファイル名を取得
→ ファイル検索を実行し、結果をクリップボードにコピー
```

### 直前の出力をコピー
```
User: git status を実行して
Assistant: [git statusの出力を表示]
User: /copy
→ 直前のgit status出力がクリップボードにコピーされます
```
