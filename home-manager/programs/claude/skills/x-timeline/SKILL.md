---

X (Twitter) のタイムラインをスクレイピングし、直近24時間の投稿の中からいいね数・RT数が多い技術系トピックを10件抽出してリストアップする。

## 引数

$ARGUMENTS

- 特定のトピックに絞る場合は指定（例: `AI`, `Rust`, `インフラ`）
- 引数を指定すると X の検索 API を使ってフォロー外を含む検索結果から抽出する
- 省略時はホームタイムライン（おすすめ）から技術系トピックを抽出する

## 処理フロー

### Step 1: スクレイパーを実行

このスキルファイルと同じディレクトリにある `scrape-timeline.ts` を実行する。
スクリプトのパスは `$SKILL_DIR` または `__dirname` 相当で解決する。

引数（トピック）が指定されている場合は検索クエリとして渡す:
```bash
<このSKILL.mdと同じディレクトリ>/scrape-timeline.ts "検索クエリ" 2>/tmp/x-timeline-log.txt
```

引数なしの場合はホームタイムラインを取得:
```bash
<このSKILL.mdと同じディレクトリ>/scrape-timeline.ts 2>/tmp/x-timeline-log.txt
```

実行結果:
- **stdout**: ツイートの JSON 配列（engagement 降順でソート済み。ホームタイムラインは直近24時間、検索は話題のポスト全期間）
- **stderr**: 進捗ログ（`/tmp/x-timeline-log.txt` に保存）

エラー時（終了コードが 0 以外）は stderr の内容を表示してスキルを終了する。

### Step 2: 技術系トピックの判定

取得した JSON 配列の各ツイートを分析し、技術的なトピックかどうかを判定する。

**技術系と判断する基準（含める）**:
- プログラミング言語、フレームワーク、ライブラリの話題
- AI / 機械学習 / LLM / エージェントに関する技術的内容
- クラウド、インフラ、DevOps、Kubernetes、Docker 等
- 開発ツール、エディタ、CLI ツールの新機能・リリース
- OSS のリリースや PR、Issue の紹介
- 技術ブログ記事・論文・カンファレンス発表
- セキュリティの脆弱性・パッチ情報
- パフォーマンスの改善・ベンチマーク
- エンジニアリングの知見・アーキテクチャの議論
- 技術的な数値・ベンチマーク・比較を含む投稿

**技術系でないと判断する基準（除外）**:
- 日常生活・グルメ・旅行・娯楽の話題
- 政治・社会問題
- 暗号資産・NFT の価格煽り（技術的考察は含める）
- 単なる宣伝・マーケティング（具体的な技術情報がない場合）
- ミーム・ジョーク（技術的な示唆がない場合）
- 求人・採用情報（技術スタックの紹介がない場合）

**引数でトピックが指定された場合**は、そのトピックに関連する投稿をさらに絞り込む。

### Step 3: Top 10 の選出

以下を考慮してランキングを決定する:

1. **エンゲージメントスコア** (`engagement = likes + retweets`) を主軸にする
2. **技術的な深さ・独自性**が高い投稿は engagement が低くても優先する
3. **同一人物の投稿**が連続する場合は多様性のため間引く（最大2件/人を目安）
4. **10件に満たない**場合は見つかった件数を全て報告する

### Step 4: 結果を出力

以下のフォーマットで日本語でレポートを出力する:

```
## X Technical Highlights（過去24時間）

**取得日時**: YYYY/MM/DD HH:MM JST
**取得件数**: N 件中 技術系 M 件

---

### 1. [投稿の内容を端的に表す日本語タイトル]

> [ツイート本文（長い場合は150文字程度に要約してもよい）]

**@author**（表示名） | ❤️ {likes} 🔁 {retweets} | {YYYY/MM/DD}
🔗 {url}

---

### 2. ...
```

引数でトピックが指定されていた場合は、タイトルにそのトピックに関連することを示す。

## エラー対処

| エラー内容 | 対処法 |
|---|---|
| Cookie DB が見つからない | Arc または Chrome をインストールして x.com にログインする |
| 認証エラー (401/403) | ブラウザで x.com を再読み込みして Cookie を更新する |
| レートリミット (429) | 数分後に再実行する |
| GraphQL エラー | scrape-timeline.ts の `GRAPHQL_QUERY_ID` を最新の値に更新する |
| 技術系投稿が0件 | タイムラインに技術系の投稿がない可能性がある。引数でトピックを絞っている場合は除外してみる |
